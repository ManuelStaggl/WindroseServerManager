    **`FetchLatestAsync(CancellationToken ct)`:**
    1. Try `GET WindrosePlusApiUrl` via `_httpFactory.CreateClient(HttpClientName)`, `Timeout = TimeSpan.FromSeconds(10)`, UserAgent + `Accept: application/vnd.github+json` (pattern from AppUpdateService).
    2. On HTTP success: parse JSON with `JsonDocument.ParseAsync`. Reject `draft: true` and `prerelease: true` with `InvalidOperationException`. Read `tag_name`, loop `assets[]`, pick the asset whose `name` equals `WindrosePlusAssetName` (case-insensitive). Extract `browser_download_url`, `size`, `digest` (nullable — log warning at Information level if absent: `_logger.LogWarning("WindrosePlus asset {Asset} has no digest field", WindrosePlusAssetName);`). Return `WindrosePlusRelease`. Write the raw JSON to `_metadataCachePath` for offline fallback BEFORE returning.
    3. On HTTP failure (HttpRequestException, TaskCanceledException from timeout): log warning with structured template, then try reading `_metadataCachePath`. If file exists, parse it and return the release (same asset-selection logic). If not, throw `WindrosePlusOfflineException("WindrosePlus release metadata unavailable and no cached copy found.", inner)`.

    **Private `FetchUe4ssLatestAsync(CancellationToken ct)`:** mirror `FetchLatestAsync` but against `Ue4ssApiUrl`. Asset selection: first asset whose `name.StartsWith("UE4SS_", OrdinalIgnoreCase)` AND `name.EndsWith(".zip", OrdinalIgnoreCase)`. Cache metadata at `_ue4ssMetadataCachePath`.

    **`InstallAsync(string serverInstallDir, IProgress<InstallProgress>? progress, CancellationToken ct)`:**
    ```
    await _installLock.WaitAsync(ct);
    try {
      Report(InstallPhase.FetchingRelease, "Lade WindrosePlus-Release-Info...");
      var wplusRelease = await FetchLatestAsync(ct);
      Report(InstallPhase.FetchingRelease, "Lade UE4SS-Release-Info...");
      var ue4ssRelease = await FetchUe4ssLatestAsync(ct);

      Report(InstallPhase.DownloadingArchive, "Lade WindrosePlus-Archiv...");
      var wplusZip = await EnsureArchiveCachedAsync(wplusRelease, _archiveCachePath, ct);
      Report(InstallPhase.DownloadingArchive, "Lade UE4SS-Archiv...");
      var ue4ssZip = await EnsureArchiveCachedAsync(ue4ssRelease, _ue4ssArchiveCachePath, ct);

      Report(InstallPhase.VerifyingDigest, "Verifiziere SHA-256...");
      var wplusSha = await VerifyDigestAsync(wplusZip, wplusRelease.DigestSha256, ct);
      var ue4ssSha = await VerifyDigestAsync(ue4ssZip, ue4ssRelease.DigestSha256, ct);

      // Temp dir on SAME volume as serverInstallDir to guarantee atomic File.Move.
      var tempRoot = Path.Combine(Path.GetDirectoryName(Path.GetFullPath(serverInstallDir))!, ".wplus-install-temp-" + Guid.NewGuid().ToString("N"));
      Directory.CreateDirectory(tempRoot);
      try {
        Report(InstallPhase.Extracting, "Entpacke WindrosePlus...");
        ZipFile.ExtractToDirectory(wplusZip, tempRoot, overwriteFiles: true);
        Report(InstallPhase.Extracting, "Entpacke UE4SS...");
        // UE4SS extracts into R5\Binaries\Win64 (per install.ps1)
        var ue4ssTarget = Path.Combine(tempRoot, "R5", "Binaries", "Win64");
        Directory.CreateDirectory(ue4ssTarget);
        ZipFile.ExtractToDirectory(ue4ssZip, ue4ssTarget, overwriteFiles: true);

        Report(InstallPhase.Installing, "Installiere Dateien...");
        AtomicMergeIntoServer(tempRoot, serverInstallDir, preserveUserConfig: true);

        Report(InstallPhase.Installing, "Kopiere LICENSE...");
        CopyLicense(tempRoot, serverInstallDir);

        Report(InstallPhase.WritingMarker, "Schreibe Versionsmarker...");
        var marker = new WindrosePlusVersionMarker(wplusRelease.Tag, DateTime.UtcNow, wplusSha, ue4ssRelease.Tag);
        File.WriteAllText(Path.Combine(serverInstallDir, VersionMarkerFileName),
          JsonSerializer.Serialize(marker, new JsonSerializerOptions { WriteIndented = true }));

        Report(InstallPhase.Complete, "WindrosePlus installiert.");
        return new WindrosePlusInstallResult(wplusRelease.Tag, serverInstallDir, marker.InstalledUtc, wplusSha, ue4ssRelease.Tag);
      } finally {
        Report(InstallPhase.CleaningUp, "Räume temporäres Verzeichnis auf...");
        try { Directory.Delete(tempRoot, recursive: true); }
        catch (Exception ex) { _logger.LogDebug(ex, "Temp dir cleanup failed at {Path}", tempRoot); }
      }
    } finally { _installLock.Release(); }
    ```

    **`EnsureArchiveCachedAsync(WindrosePlusRelease release, string cachePath, CancellationToken ct)`:**

    2. Try download: `HttpCompletionOption.ResponseHeadersRead` → `await response.Content.CopyToAsync(fs, ct)` — write to `cachePath + ".tmp"`, then `File.Move(tmp, cachePath, overwrite: true)` (atomic on same volume).
    3. On HttpRequestException/TaskCanceledException: if `File.Exists(cachePath)` log warning and return `cachePath` (cache fallback). Else throw `WindrosePlusOfflineException`.

    **`VerifyDigestAsync(string path, string? expectedDigest, CancellationToken ct)`:** per Research §"SHA-256 verification". If `expectedDigest` null/empty, log warning and compute+return the actual SHA-256 hex. Otherwise strip `"sha256:"` prefix (case-insensitive), compute actual via `SHA256.HashDataAsync(FileStream, ct)`, compare case-insensitively. On mismatch throw `WindrosePlusDigestMismatchException($"Archive SHA256 mismatch at {path}: expected {expected}, actual {actual}")`. Returns the actual SHA-256 hex string (used for marker file).

    **`AtomicMergeIntoServer(string tempRoot, string serverDir, bool preserveUserConfig)`:** per Research §"Atomic copy preserving user config".
    User-owned relative paths (case-insensitive, skip overwrite only if destination already exists):
    ```csharp
    private static readonly string[] UserOwnedRelativePaths =

        "windrose_plus.json",
        "windrose_plus.ini",

    };
    ```

    - `File.Move(sourceFile, dest, overwrite: true);` (atomic since same volume).

    **`CopyLicense(string tempRoot, string serverDir)`:** find case-insensitive `LICENSE` or `LICENSE.txt` at `tempRoot` root, copy to `Path.Combine(serverDir, BundledLicenseFileName)` (overwrite). If missing, log warning, do not throw.
